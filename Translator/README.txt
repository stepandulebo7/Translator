Android-переводчик

Простое и удобное приложение для перевода текста на Android. Позволяет пользователям вводить текст, выбирать исходный и целевой языки и получать перевод в реальном времени с помощью Google Translate API.

	Основные возможности 

Мгновенный перевод: Перевод текста между множеством языков.
Динамическая загрузка языков: Список поддерживаемых языков автоматически загружается из локального файла `languages.json`.
Смена языков: Возможность быстро поменять местами исходный и целевой языки одной кнопкой.
Копирование в буфер обмена: Копирование переведенного текста одним нажатием.
Интуитивный интерфейс:
Динамические подсказки, которые меняются в зависимости от выбранного языка.
Анимация во время процесса перевода для улучшения пользовательского опыта.
Ограничение ввода: Ввод текста ограничен 500 символами для предотвращения слишком длинных запросов.
Обработка ошибок: Встроена обработка ошибок сети и API для информирования пользователя о проблемах.

	Технологии и библиотеки (Technologies and Libraries)

Язык программирования: Java
Платформа: Android
API для перевода: Google Translate API v2
Сетевые запросы: [OkHttp](https://square.github.io/okhttp/) - для выполнения асинхронных HTTP-запросов.
Работа с JSON: [Gson](https://github.com/google/gson) - для парсинга JSON-ответов от API и данных из `languages.json`.

	Как это работает (How it Works)

Вся основная логика сосредоточена в классе `MainActivity`.

	1. Загрузка языков
При запуске приложение выполняет следующие действия:
- Читает файл `assets/languages.json`.
- С помощью библиотеки `Gson` парсит JSON-массив в список объектов `Language`.
- Заполняет два `Spinner` (выпадающих списка) названиями языков, устанавливая по умолчанию "Русский" как исходный и "Английский" как целевой.

	2. Процесс перевода
1.  Пользователь вводит текст, выбирает языки и нажимает кнопку "Перевести".
2.  Приложение проверяет, не пустое ли поле ввода и не совпадают ли выбранные языки.
3.  Для пользователя отображается анимированная подсказка "Текст переводится...".
4.  Формируется JSON-объект с текстом и кодами языков.
5.  С помощью `OkHttp` создается и асинхронно выполняется POST-запрос к `https://translation.googleapis.com/language/translate/v2`.
6.  **API-ключ** добавляется как параметр в URL запроса.
7.  После получения ответа, он парсится с помощью `Gson`, и извлеченный переведенный текст отображается на экране.

	3. Пользовательский интерфейс (UI)
-   UI построен на стандартных компонентах Android (`EditText`, `TextView`, `Spinner`, `Button`, `ImageButton`).
-   Динамические элементы:
    -   Подсказка в поле переведенного текста (`"Translated text"`, `"Переведённый текст"`) меняется в зависимости от выбранного целевого языка.
    -   Кнопка "Копировать" становится видимой только после успешного перевода.
    -   Во время ожидания ответа от API поле результата очищается и запускается плавная анимация `AlphaAnimation`.

	Установка и запуск (Installation and Setup)

1.  Клонируйте репозиторий.
2.  Откройте проект в Android Studio.
3.  ВАЖНО: В файле `app/src/main/java/com/example/kursovaay/MainActivity.java` необходимо заменить демонстрационный API-ключ для Google Translate API на ваш собственный.

    ```java
    // Найдите эту строку и вставьте свой ключ
    private static final String API_KEY = "ВАШ_API_КЛЮЧ";
    ```
4.  Соберите и запустите приложение на эмуляторе или реальном устройстве.

	Структура проекта (Project Structure)

-   `app/src/main/java/com/example/kursovaay/MainActivity.java`: Единственная активность приложения, содержащая всю бизнес-логику, обработку UI и сетевые вызовы.
-   `app/src/main/res/layout/activity_main.xml`: XML-файл разметки для `MainActivity`.
-   `app/src/main/assets/languages.json`: JSON-файл со списком поддерживаемых языков и их ISO 639-1 кодами.
